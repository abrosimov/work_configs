SessionHandler::open

(PHP 5 >= 5.4.0)

SessionHandler::open — Initialize session

Description

public int SessionHandler::open ( string $save_path , string $sessionid )

Re-initialize existing session, or creates a new one. Called from session_start().

Parameters

save_path

The path where to store/retrieve the session.

sessionid

The session id.

Return Values

The return value of the session storage (usually 0 on success, 1 on failure).
