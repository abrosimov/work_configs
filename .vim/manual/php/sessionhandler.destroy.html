SessionHandler::destroy

(PHP 5 >= 5.4.0)

SessionHandler::destroy — Destroy a session

Description

public int SessionHandler::destroy ( string $sessionid )

Destroys a session. Called by session_regenerate_id(), session_destroy() and when session_decode() fails.

Parameters

sessionid

The session ID being destroyed.

Return Values

The return value of the session storage (usually 0 on success, 1 on failure).
