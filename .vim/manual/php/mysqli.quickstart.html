This quickstart guide shall help choosing an PHP MYSQL API and help getting familiar with it.

The quickstart gives an overview on the mysqli extension. Code examples are given for all major aspects of the API. Database concepts are explained to the degree needed for presenting MySQL specifics.

Familarity with the PHP programming language, the SQL language and basic knowledge of the MySQL server is assumed.
